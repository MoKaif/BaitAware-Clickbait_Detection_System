What is BaitAware ?

In the digital age, the prevalence of clickbait poses challenges for users, content creators, and platforms alike, leading to misinformation and frustration. Our project introduces a Clickbait Detection System that combines advanced machine learning and image processing techniques to accurately identify clickbait content. Through a user-friendly web application interface, users can make informed decisions about the content they engage with online. We delve into the technical details of text and image analysis, highlighting the significance of our system in promoting a trustworthy online environment.

BaitAware Objectives

  • Research and Model Evaluation: Conduct research on clickbait detection methodologies and evaluate various machine learning models for both text and image classification.
  • Dataset Acquisition and Preparation: Gather datasets for text and image classification, preprocess the data, and create a unified dataset for training the models.
  • Model Training and Selection: Train machine learning models, including Multinomial Bayes for textual classification and Convolutional Neural Networks (CNN) for image classification. Select the best-performing models for integration into the ensemble.
  • OCR Integration: Implement Optical Character Recognition (OCR) to extract textual data from images and append it to the dataset, enhancing the text classification model's performance.
  • Ensemble Model Development: Develop an ensemble model that aggregates predictions from the text and image classification models to enhance clickbait detection accuracy.
